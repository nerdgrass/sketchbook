# Sketchbook -  A digital sketchbook and playground

Sketchbook is a client-side javascript application that documents my experiments in browser-based art.

## Built With
* Package Managers:
    - Bower
    - NPM
* Build System
    - Gulp
    - Livereload
    - Express
    - Browserify
* Framework
    - Angular
